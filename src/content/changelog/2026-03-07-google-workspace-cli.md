Installed [Google Workspace CLI](https://github.com/googleworkspace/cli) — can now go from a vault markdown file to a Google Sheet, Slides deck, or formatted email in one command. Previously, the only path to Google Workspace was browser automation — slow, brittle, breaks when the UI changes. The new scripts talk directly to Google's API, so a markdown table becomes a formatted spreadsheet in seconds, not minutes of clicking through Chrome.
